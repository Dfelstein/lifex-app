Booking Conversion Tracking
⚡ Action needed
  • Create /booking-confirmed page in WordPress
  • In Acuity → Business Settings → Scheduling Page Settings
  • Set confirmation redirect to: xgym.com.au/booking-confirmed
  • In GA4 → Mark this page as a conversion event
  • In PixelYourSite → Mark as Meta conversion event
This is the most important setup step — without it we can't track which ads drive bookings.
