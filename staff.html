New Client
Two steps to set up a new client account
Step 1 — Create Login Account in Supabase
1. Open supabase.com → your project → Authentication → Users
2. Click Add user → Create new user
3. Enter the client's email + a temporary password (e.g. LifeX2025!)
4. Tick "Auto Confirm User" so they can log in immediately
5. Copy their UUID (shown after creating) and paste it below
